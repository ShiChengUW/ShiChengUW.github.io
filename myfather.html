A braid that can be tightened and loosend easily

The follwoing is a braid that can be used to various circumstances, accoding to my father. There are some properties for this braid. For instance, it can be completely loosed to a straight line by pulling some ends (A and F) of the rope. However, if pulling one other part (E), the rope becomes tight.

The string can be very tight between two trees, so it can be used to save people during a flood. This braid can also be used to tie legs of cattles, so my father call it 捆牛绳 or an overvalued long name "三位一体两松一固的锁扣". One can also pull up oneself when climbing a cliff. These applications are expectations and have passted any test, so I cannot be sure if the braid really works or safe.
